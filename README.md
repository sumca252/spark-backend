# spark-backend
[![test](https://github.com/sumca252/spark-backend/actions/workflows/test.yml/badge.svg)](https://github.com/sumca252/spark-backend/actions/workflows/test.yml)

Project repo for the course Vteam at BTH University.

### Install 
```bash
npm install
```
### Run in dev mode
```bash
npm run dev
```

### Run in prod mode
```bash
npm start
```

### Test 
```bash
npm test
```

### Lint
```bash
npm run eslint
```

### Format
```bash
npm run fix
```
